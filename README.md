# ABBA

![img](img.png)

### Rules:

Write a function with two input params - string and int. The function returns string without every consecutive N character of the string defined by int.

### Examples:

#### A

Input: 
```
"aaab", 2
```
Result: 
```
"aab"
```
#### B
Input:
```
"aabb", 1
```
Result: 
```
"ab"
```
#### C
Input: 
```
"aabbaa", 1 
```
Result: 
```
"aba"
```
#### D
Input: 
```
"aabbbbaa", 1 
```
Result: 
```
"abba"
```
